# Usage
<!-- usage -->
```sh-session
$ npm install -g aem
$ aem COMMAND
running command...
$ aem (-v|--version|version)
aem/0.0.0 darwin-x64 node-v10.10.0
$ aem --help [COMMAND]
USAGE
  $ aem COMMAND
...
```

## `aem help [COMMAND]`

display help for aem

```
USAGE
  $ aem help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
