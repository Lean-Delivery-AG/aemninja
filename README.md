# Usage
<!-- usage -->
```sh-session
$ npm install -g aemninja
$ aemninja COMMAND
running command...
$ aemninja (-v|--version|version)
aemninja/0.0.0 darwin-x64 node-v10.10.0
$ aemninja --help [COMMAND]
USAGE
  $ aemninja COMMAND
...
```

## `aemninja help [COMMAND]`

display help for aemninja

```
USAGE
  $ aemninja help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
